# value-converter

Cryptocurrency Converter (CustomTkinter)
Description

This Python application is a simple cryptocurrency converter with a graphical interface built using CustomTkinter. It allows users to convert between BTC, ETH, USDT, and UAH using predefined exchange rates. The app features input validation, real-time conversion, and a clean, modern interface.

Features

Convert between BTC, ETH, USDT, and UAH

User-friendly GUI with CustomTkinter

Input validation to prevent invalid entries

Displays results with appropriate decimal precision
